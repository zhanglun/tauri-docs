Benchmarks

All benchmarks run on Github Actions on ubuntu-latest matrix. We measure various metrics of the following applications:

  • tauri_cpu_intensive
  • tauri_hello_world
  • tauri_3mb_transfer
  • wry_cpu_intensive
  • wry_hello_world
  • wry_custom_protocol
  • electron_cpu_intensive
  • electron_hello_world
  • electron_3mb_transfer
